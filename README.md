## About This Project

This repository hosts my Professional Identity Infographic, created as part of the COP 4910 course assignment at the University of Central Florida. The infographic tells my story as an emerging IT professional through six required sections covering my professional brand, motivation, coursework, integrated experience connection, future vision, and personal background.

The infographic is built as a single-page HTML file and deployed via GitHub Pages.
